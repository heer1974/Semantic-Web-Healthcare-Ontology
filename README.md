# Semantic Web Healthcare Ontology

## 📌 Overview

This project focuses on the design and implementation of a Healthcare Ontology using Semantic Web technologies. The ontology was developed in Protégé using RDF and OWL to represent healthcare concepts, relationships, and patient information. SPARQL queries were used to retrieve and analyse semantic data efficiently.

---

## 🚀 Technologies Used

- RDF (Resource Description Framework)
- OWL (Web Ontology Language)
- SPARQL
- Protégé
- Semantic Web

---

## ✨ Project Features

- Designed a healthcare ontology using Protégé
- Created classes, subclasses, and class hierarchies
- Implemented object properties and data properties
- Added individuals (instances) to the ontology
- Established semantic relationships between healthcare entities
- Executed SPARQL queries for information retrieval
- Validated ontology consistency using reasoning tools
